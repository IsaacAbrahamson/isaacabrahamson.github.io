Inventory App

An Android app used for tracking inventory levels. This artifact was created using Java and demonstrates my abilities in software design and development.

Artifact Link

  • GitHub Repository

Overview

My artifact for the software engineering and design portion is an inventory management android application. This application lets users add inventory items and track their quantity levels. When the levels go below a warning, the user can receive a notification. This was created as part of the mobile application development class at SNHU.

Justification and Enhancement

I selected this item because it showcases building a complete application from start to finish from the ground up. Reflection

The biggest challenge when working on this application is figuring out the Android sdk and development. I had to read through a lot of android development documentation to understand how to send push notifications, and I also had to read a lot on some of the view components to be able to capture input events correctly.
