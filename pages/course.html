Course Finder

A command line application for viewing and searching courses and their requirements using different data structures. This artifact was created using C++ and demonstrates my abilities in data structures and algorithms.

Artifact Link

  • GitHub Repository

Overview

This is a command line application I built for the CS-300 Data Structures and Algorithms class. It contains a simple application to load a list of courses from a file into a data structure and then print the whole list as well as specific courses and their perquisites.

Justification and Enhancement

I selected this item because it shows how to use several different data structures to accomplish the same task. Additionally, it also shows the creation of a complex data structure from scratch with the Binary Search Tree. For this assignment, I made several improvements. My first improvement was to take some of the description from a word file into the project readme. My next improvement was to improve the sample data from about 10 courses to 200. My next improvement was to add timings to the method calls so the difference could be seen between the chosen data structure. My next improvement was adding vector and hash table data structures in addition to the binary search tree that could be chosen from. Finally, the addition of the extra data structures made the code pretty complex, so I created an abstract structure class that all three data structures implemented so that the one course menu could be greatly simplified and it uses the abstract methods based on whichever structure the user chooses at runtime.
